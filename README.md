#  Tic Tac Toe Game

A fun and interactive Tic Tac Toe game built with HTML, CSS, and 
JavaScript that allows two players to play against each other 
in the browser.

##  Features

- **Two Player Mode** — Play with a friend on the same device
- **Winner Detection** — Automatically detects win conditions
- **Draw Detection** — Identifies when the game is a draw
- **Restart Button** — Reset the game anytime
-  Fully **Responsive Design** for desktop
-  Clean and attractive **UI with animations**
-  **Score Tracking** — Keeps track of wins for both players
